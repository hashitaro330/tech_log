# 塾の活動(FY23 下期)

## テーマ

「クラウドネイティブな環境におけるモニタリングツールの比較と構築ノウハウの整理」

### 背景

- クラウドやオンプレ、コンテナといった形で、アプリケーションを動かす基盤部分は多様化している。それらを包括的にモニタリングするツールに関しても開発が活発であり、商用システムである Datadog や、OSS である Prometheus 等があげられる。
- マルチクラウド環境での基盤の技術検証を行う機会が増えたことに伴い、今後複雑な環境下での基盤のモニタリングが求められる。それに伴い、今後の PoC 案件での適切なモニタリングツールの選定と利活用するためのノウハウを獲得したい。

### 目的

- Datadog が自組織の PJ に利活用できそうか評価すること
- Java の実装スキルを身に着け、基盤だけでなくアプリケーションも理解できるようになること
